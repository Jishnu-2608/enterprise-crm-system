# Enterprise CRM System

## 📌 Project Overview

Enterprise CRM System is a web application developed using React.js for managing customer relationships. It allows users to manage leads and customers through an intuitive dashboard with persistent local storage.

## 🚀 Features

* User Login
* Dashboard
* Lead Management (Add, Edit, Delete)
* Customer Management (Add, Edit, Delete)
* Lead Deal Stages
* Search Leads & Customers
* Lead Stage Filter
* Export Leads to CSV
* Export Customers to CSV
* Local Storage Support
* Responsive User Interface

## 🛠️ Technologies Used

* React.js
* Vite
* JavaScript
* HTML
* CSS
* LocalStorage

## 📂 Project Structure

* Dashboard
* Login
* Leads
* Customers
* Components

## ▶️ How to Run

1. Clone the repository
2. Install dependencies
3. Run:

npm install

npm run dev
